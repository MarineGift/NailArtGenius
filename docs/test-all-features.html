🧪 Connie's Nail 전체 기능 테스트

모든 기능을 체계적으로 테스트할 수 있는 페이지입니다.

📱 메인 사이트로 이동

📋 테스트 방법

1. 아래 "메인 사이트로 이동" 버튼을 클릭하세요

2. 각 기능별로 순서대로 테스트하세요

3. 문제가 발생하면 브라우저 개발자 도구(F12)에서 Console 오류를 확인하세요

🌍 언어 전환 테스트

  • ✅ 기본 상태: 영어로 표시 Ready
  • 🇰🇷 Korean 선택 → 한글 변환
  • 🇯🇵 Japanese 선택 → 일본어 변환
  • 🇪🇸 Spanish 선택 → 스페인어 변환
  • 🇺🇸 English 선택 → 완전 복원

테스트 위치: 우측 상단 드롭다운 메뉴

확인 항목: 네비게이션, 버튼, 폼 필드 모두 번역

🤖 AI 네일아트 테스트

  • 📍 네비게이션에서 "🤖 AI Nail Art" 클릭
  • 📷 촬영 가이드 3개 예시 확인
  • 📱 사진 업로드 (Select Photos)
  • 🔬 "Analyze with AI" 버튼 클릭
  • 📊 분석 진행률 표시 (0%-100%)
  • 🎨 10개 디자인 생성 확인
  • 📄 PDF 다운로드 테스트

예상 결과: 3초 후 10개 맞춤 디자인 + PDF 버튼

📅 예약 시스템 테스트

  • 📍 "Booking" 메뉴 클릭
  • 📝 모든 필드 영어 표시 확인
  • ✍️ Name: "Enter your name"
  • 📞 Phone, Email 필드 확인
  • 📝 Notes: "Any special requests..."
  • 🕐 시간 선택: "Choose a time..."
  • ✅ "Confirm Booking" 버튼 테스트

중요: 한글이 전혀 표시되지 않아야 함

🔧 문제 해결 가이드

  • 사진 업로드 안됨: JPG, PNG 파일만 가능
  • 언어 전환 안됨: 페이지 새로고침 후 재시도
  • AI 분석 버튼 안보임: 최소 1장 사진 업로드 필요
  • JavaScript 오류: F12 → Console 탭에서 오류 확인
  • 모바일 테스트: 브라우저 개발자 도구에서 모바일 뷰 확인
