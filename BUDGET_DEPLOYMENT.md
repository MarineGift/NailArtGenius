## 🚀 배포 단계별 가이드

### 1단계: Deploy 버튼 찾기
```
Replit 프로젝트 화면에서:
1. 상단 오른쪽 "Deploy" 버튼 클릭
   또는
2. 좌측 사이드바 "Deployments" 클릭
```

### 2단계: 배포 설정
- **무료**: "Deploy" → "Static Deployment" 선택
- **유료**: "Deploy" → "Autoscale Deployment" 선택

### 3단계: 도메인 설정 (유료만)
```
1. "Custom Domain" 섹션에서
2. "nolsae.com" 입력
3. DNS 설정 안내 따라하기
```

### 4단계: DNS 설정 (도메인 구매 후)
```
도메인 관리 페이지에서:
A     @     [Replit 제공 IP]
CNAME www   nolsae.com
```
