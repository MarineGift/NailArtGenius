## 🌐 배포 방법

### 자동 배포 (권장)
```bash
cd docs
./deploy-github.sh
```

### 수동 배포
1. GitHub에서 `connienail` repository 생성
2. docs/ 폴더 내용을 repository에 업로드
3. Settings → Pages에서 main branch 선택
4. https://marinegift.github.io/connienail/ 접속
